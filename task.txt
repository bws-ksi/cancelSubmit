Создать страницу с формой. Можно переделать (сохранив) предыдущую форму.
Стилизация минимальная. Основное события.

Поля: First name, last name, age, email, password, confirm password
Радио кнопки: male, female.
Кнопки: cancel, submit.

На кнопку cancel повесить обработчик - очистка всех полей и радио кнопок.
Не забудьте про preventDefault()
На кнопку submit повесить обработчик - собрать все данные с полей и создать из них объект с помощью конструктора (класс) с именем = lastName.
Затем сохранить этот объект в localStarage с ключом lastName.
